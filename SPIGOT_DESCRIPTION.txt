[SIZE=5][B][COLOR=#4A90E2]Description[/COLOR][/B][/SIZE]

OnlineMonitor is a powerful plugin for tracking and analyzing player activity on your Minecraft server. Get detailed statistics, beautiful graphs, and Discord notifications in real-time!

Whether you're running a small community server or a large network, OnlineMonitor provides comprehensive insights into your player base with an intuitive web dashboard and seamless Discord integration.

[SIZE=5][B][COLOR=#4A90E2]Commands[/COLOR][/B][/SIZE]

[B]In-Game Commands:[/B]
[CODE]
/online - Show basic online statistics
/online stats - Show detailed server statistics
/online top - Show top 10 players by join count
/online player <name> - Show statistics for a specific player
/online hourly [days] - Show average online by hour (default: 7 days)
/online daily [days] - Show average online by day (default: 7 days)
/online weekday [weeks] - Show average online by weekday (default: 4 weeks)
/online peak [days] - Show peak activity hours (default: 7 days)
[/CODE]

[B]Discord Slash Commands:[/B]
[CODE]
/online - Show current online players
/stats - Show detailed server statistics
/top - Show top 10 players by activity
/player <nickname> - Show statistics for a specific player
[/CODE]

[SIZE=5][B][COLOR=#4A90E2]Permissions[/COLOR][/B][/SIZE]

[CODE]
onlinemonitor.* - All plugin permissions
onlinemonitor.basic - Basic /online command (default: true)
onlinemonitor.stats - Access to extended statistics (default: true)
onlinemonitor.admin - Admin permissions (default: op)
[/CODE]

[SIZE=5][B][COLOR=#4A90E2]REST API[/COLOR][/B][/SIZE]

OnlineMonitor provides a comprehensive REST API for third-party integrations:

[CODE]
GET /api/stats - General server statistics
GET /api/online - Current online count and player list
GET /api/players?limit=N - Top N players by activity
GET /api/players?name=PlayerName - Statistics for a specific player
GET /api/snapshots?type=hourly&days=7 - Hourly average values
GET /api/snapshots?type=daily&days=30 - Daily average values
GET /api/snapshots?type=weekday&weeks=4 - Weekday averages
GET /api/snapshots?type=peak&days=7 - Peak activity hours
[/CODE]

[B]Example Response (JSON):[/B]
[CODE]
{
  "maxOnline": 42,
  "uniquePlayers": 156,
  "totalSessions": 1247,
  "activeSessions": 8,
  "totalPlaytime": 125678900
}
[/CODE]

[SIZE=5][B][COLOR=#4A90E2]Quick Start Guide[/COLOR][/B][/SIZE]

[B]1. Basic Installation:[/B]
[LIST=1]
[*] Download OnlineMonitor.jar
[*] Place it in your server's plugins/ folder
[*] Start the server (config.yml will be created)
[*] Enjoy basic statistics tracking!
[/LIST]

[B]2. Discord Setup (Optional):[/B]
[LIST=1]
[*] Create a Discord bot at [URL='https://discord.com/developers/applications']Discord Developer Portal[/URL]
[*] Enable MESSAGE CONTENT INTENT in bot settings
[*] Copy the bot token
[*] Get your channel ID (enable Developer Mode in Discord)
[*] Add token and channel ID to config.yml
[*] Restart the server
[/LIST]

[B]3. Web Dashboard Setup (Optional):[/B]
[LIST=1]
[*] Set web-panel.enabled: true in config.yml
[*] Configure the port (default: 8080)
[*] Restart the server
[*] Access at http://your-server-ip:8080
[/LIST]

[B]Detailed Setup Guide:[/B] [URL='https://github.com/Vogulev/Minecraft_online_monitor_plugin']View on GitHub[/URL]

[SIZE=5][B][COLOR=#4A90E2]Configuration[/COLOR][/B][/SIZE]

[SPOILER="config.yml Example"]
[CODE=YAML]
# Welcome message for joining players
welcome-message: "Welcome to the server, %player%!"

# Logging settings
log-unique-joins: true
log-join-quit: true

# Database (sqlite or mysql)
database:
  type: sqlite
  mysql:
    host: localhost
    port: 3306
    database: minecraft_stats
    username: root
    password: password

# Snapshot interval (in minutes)
snapshot-interval-minutes: 5

# How many days to keep snapshots
snapshot-days-to-keep: 30

# Timezone offset from UTC
timezone-offset: +3

# Discord integration
discord:
  enabled: false
  bot-token: "YOUR_BOT_TOKEN_HERE"
  channel-id: "YOUR_CHANNEL_ID_HERE"
  notifications:
    player-join: true
    player-quit: true
    new-player: true
    new-record: true
    server-start: true
    server-stop: true

# Web dashboard
web-panel:
  enabled: false
  port: 8080
[/CODE]
[/SPOILER]
